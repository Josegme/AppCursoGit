Welcome

La programación es el proceso de crear instrucciones para que una computadora ejecute tareas. Esto implica diseñar, codificar, mantener y proteger el código fuente de los programas.

Objetivos de la programación: Crear software que se pueda ejecutar en computadoras, programas o plataformas Interactuar con computadoras para realizar investigaciones, diseñar productos y servicios, administrar finanzas, y más Simular fenómenos, analizar datos y modelar sistemas complejos Enseñar a las máquinas a tomar decisiones
